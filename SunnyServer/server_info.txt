Our server is at https://sunny-scanner.herokuapp.com/

Heroku will detect our server as a Node.js if we don't specify that it is python so we need to
configuer heroku to use python
> heroku config:set BUILDPACK_URL=https://github.com/heroku/heroku-buildpack-python

To delpoy app to heroku remote use 
> git push heroku master

To delopy from a non mater branch use 
> git push heroku <branch-name>:master
