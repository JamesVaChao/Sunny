Go to https://www.python.org/ and install python

To install all dependencies 
>pip install -r requirements.txt

The command
>pip freeze 
will print all intalled python modules, if you make a change to the modules used, 
run this command to check the version of the module installed and then add the 
module to the requirements.txt

For testing locally run in app.py set and test from postman
app.run(debug=True)  

Run in your command line using:
>python app.py

To test from expo app set
app.run(host='0.0.0.0')
